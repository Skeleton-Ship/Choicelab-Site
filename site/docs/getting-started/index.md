## System requirements

To use Choicelab, you'll need either:

- A Mac running macOS 13.5 or higher
- A PC running Windows 11, or Windows 10 version 22H2

Choicelab isn't supported on touch devices like iPhones or iPads.

While other apps aren't strictly required to use Choicelab, you may find it useful to also have:

- An audio editor, such as GarageBand (Mac) or Audacity (cross-platform)
- A video editor, like iMovie (Mac) or DaVinci Resolve (cross-platform)

## Download & install

You can download the latest version of Choicelab [from this site](/), or from Choicelab's [GitHub Releases](https://github.com/Skeleton-Ship/Choicelab/releases).

Once it's installed, drag the Choicelab app icon to your **Applications** folder (Mac) or run the installer (Windows). That's pretty much it!
