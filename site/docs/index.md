---
title: Choicelab Library
subtitle: Learn how to create multimedia-rich interactive narratives with Choicelab.
---

If you've ever been interested in making interactive stories or small games, there's a chance you've been introduced to tools like [Twine](https://twinery.org) or [Bitsy](https://www.bitsy.org).

These tools are popular because they're approachable, easy to use, fun, and _specific_. There are big, monolithic, many-featured tools that can make the things those tools do—but they typically have a steep learning curve, and can be expensive.

A good tool, the kind that makes you excited to create things, is usually a specific one. Choicelab's primary goal is to be **a useful tool for developing multimedia-rich interactive narratives**. Instead of hypertext, Choicelab uses _audio_ and _video_ as the foundation for its narratives.

There are lots of textual interactive fiction tools that let you tell great stories. But if you've ever wanted to make something in between films, serialized audio, and games, we're building Choicelab for you.
