Thanks for trying out Choicelab! The couple of us making it are happy you're here.

If you've ever been interested in making interactive stories or small games, there's a chance you've been introduced to tools like [Twine](https://twinery.org) or [Bitsy](https://www.bitsy.org).

These tools are popular because they're approachable, fun, and _specific_. There are big, monolithic, many-featured tools that can make the things those tools do—but they typically have a steep learning curve, and can be expensive.

A good tool, the kind that makes you excited to create things, is usually a specific one. Choicelab's primary goal is to be **a tool for developing multimedia-rich interactive stories**. Instead of textual interactive fiction, Choicelab uses _audio_ and _video_ as the building blocks for its narratives.

There are lots of textual interactive fiction tools that let you make great narrative experiences. But if you want to tell stories using motion and sound design—that perhaps bridge the gap between films, serialized audio, and games—we're building Choicelab for you.
